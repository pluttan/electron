Для добавления библиотек необходимо выполнить следующие действия:

1. Скопировать файлы из папки расположения данного файла в папку MC12/LIBRARY,
2. Найти в папке MC12/LIBRARY файл NOM.LIB и открыть его в текстовом редакторе,
3. Добавить в файл NOM.LIB приведенные ниже строки текста и сохранить файл.

.lib "Rus_d.lib"
.lib "Sovdiod.lib"

.lib "Rus_q.lib"
.lib "Sovbipol.lib"
.lib "Sovpol.lib"
